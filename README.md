# SmartCart_Clustering
E-commerce Customer Segmentation System 
- Initally Done on Jupyter Labs
- SmartCart_Clustering is an E-commerce Customer Segementation System where i performed analysis, EDA, feature engineering on the dataset and implemented K-means and Agglomerative Clustering algorithmn to create clusters of customers to help SmartCart (e-commerce bussiness) with marketing and engagement strategies.
